# Overview
This program reads a CSV file containing a list of students, stores the data in a dynamic array(Vector class) and provides the following functionalities:

1.Sort students by student number (n)
2.Sort students by first name (f)
3.Sort students by last name (l)
4.Search for a student by first name (s)

# Program output

 'f' Sort by first name
 'l' Sort by last name
 's' Search the student by first name
